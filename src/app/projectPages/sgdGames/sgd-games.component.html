Student Game Developers Games

Student Game Developers is a student-run club at UVA where students can pitch ideas for games and lead teams to develop said idea throughout the semester. It brings together programmers, artists, and writers to make every game as fleshed out and complete as possible. I have been a part of this club since my first semester in college and have enjoyed every project I've been a part of and learning how to use the Unity software. Check out these games I've contributed to so far.

A Bard's Tale (Spring 2018 - Spring 2019)

Skills Used: Unity, C#

I've been a part of this game for three semesters and it's been a different idea each semester.

The initial idea was a game where you played as a bard travelling the land in search on companions to vanquish evil. However, the twist is that your only controls are the chords that you play. How you play the chords will affect how your companions feel and act in battle. Sequences of chords could make allies attack or flee. I worked on programming the system to judge qualities of music that our team lead came up with, the general UI of the game, and setting up the class structure that the characters and enemies will be based around. The github repository can be found here.

The next semester, the leads must have realized that their initial idea was too big to be created in a single semester, so they changed the idea to have a bard exploring a dungeon using a magical lute to cast spells. The player still controlled the notes that the bard would play, and different sequences of notes would trigger different spells. Again, I chose to work on the mechanics dealing with the music and determining what notes to play at each key-press. The new github repository can be found here.

For the final semester, I worked primarily on translating some custom animation scripts into the built-in Unity animation controllers. I had never tackled animation before, so this was a great learning experience to work in a different component of Unity. Link to Github repository here:

Dusk Patrol (Fall 2017)

Skills Used: Unity, C#

Dusk Patrol is a shoot 'em up game where you play as a World War II fighter shooting through hordes of incoming planes as well as a giant blimp at the end. The game introduces a time-travel mechanic where the player can move backward in time and work with their previous self to defeat the waves of enemies. I worked on controls for moving and shooting for the player as well as the attacks for the boss-blimp. The github repository can be found here.

Kin (Fall 2016 - Spring 2017)

Skills Used: Unity, C#

Kin is a game where you play as a Spanish Monk exploring a new world vanquishing Mayan- and Aztec-inspired Gods, talking to villagers, and collect resources all within a time limit. I primarily worked on programming the behaviors of the enemies and creating the bosses for the player to fight. More information on about the game can be found here. The github repository can be found here.

This was the first game I worked on as part of SGD and spanned my entire first year of college. Through working on this game, I learned a lot about how the Unity framework worked and how a typical game-building team worked.
